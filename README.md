# Comics Master

Полнофункциональная система управления контентом для каталогизации событий из мира комиксов. Проект предоставляя динамический интерфейс для пользователей и защищенную панель администратора с блочным редактором контента.

## Основные возможности

* **Динамический каталог:** Главная страница с фильтрацией по вселенным (Marvel, DC, Both) и поиском.
* **Блочный редактор:** Панель администратора позволяет конструировать страницы информации, чередуя текстовые абзацы и изображения в произвольном порядке.

## Технологический стек

* **Frontend:** HTML, CSS, JavaScript.
* **Backend:** Node.js.
* **База данных:** MySQL.

## Структура базы данных

Проект использует две основные таблицы для реализации гибкого контента:

1. `events`: Хранит общую информацию о карточке (название, дата, превью).
2. `event_blocks`: Хранит составные части страницы (текст или URL изображения) с привязкой к ID события.

## Установка и запуск

### 1. Требования

* Node.js (версия 16.9.0 или выше рекомендуется).
* MySQL Server.

### 2. Настройка базы данных

Создайте базу данных и таблицы, используя `dump.sql`


### 3. Настройка сервера

В файле `server.js` обновите параметры подключения к MySQL:

```javascript
const db = mysql.createPool({
    host: 'localhost',
    user: 'root',
    password: 'YOUR_PASSWORD',
    database: 'comics_db'
});

```

### 4. Запуск приложения

Установите зависимости и запустите сервер:

```bash
npm install express mysql2 cors
node server.js

```

Приложение будет доступно по адресу: `http://localhost:3000`

## Использование административной панели

1. Перейдите по адресу `http://localhost:3000/admin.html`.
2. Введите стандартные учетные данные (логин: `admin`, пароль: `admin123`).
3. Используйте конструктор для добавления новых событий или редактирования существующих.
